Start Archiving

1. Disable Sleep Mode

Disable System Sleep when using LTFS
macOS systems can corrupt an LTO tape cartridge if the system goes to sleep or hibernates while an offload is in process. 

To turn off sleep mode macOS
Go to
System Preferences -> Energy Saver
and Set “Turn display off after:” to Never 

2. Get the System Ready

· Turn on the mTape if it's not on.
· Open Yoyotta.

You can access Yoyotta by hovering the mouse over the bottom of the screen. Mac's dock will pop up displaying Yoyotta's icon.

x

3. Load the LTO tape

1) Make sure mTape is powered on and the Green Ready Light is illuminated. Do not try to insert a tape cartridge before the Green Ready Light is on and steady.
2) Gently, insert the cartridge into the LTO drive. The cartridge goes in with the arrow pointing upward.
4) When the cartridge is loaded, the Green Ready Light will glow steady green. 

4. Verify all files are in the Project Folder

Before you archive a project, take a look at its folder from the finder, to make sure all assets, as well as project files are included in the main folder.
This is a common Folder structure for one of Spang’s projects. 

5. Drag and Drop
the Project Folder

Drag and drop the Project Folder you want to archive into Yoyotta Source Panel (Left side).

Project Folders are usually named with the following structure: aaa_bbbbbbbbbbbb_cccccc_ddddd
a=Agency   b=Code
c=Client   d=Project
Yoyotta’s Project Browser will open automatically. 

6. Create a Project

Inside the Project Browser, look for the Collection field and choose SPANG_POST. A Collection is a library that contains Projects.
After choosing SPANG_POST, add a New Project by clicking on the “+” (plus sign) and name it on the field right next. Use the name of the folder or drive you are archiving. The frame rate can stay at 23.98. Finally, exit the Project Browser and drag other additional Project Folders if you need to. 

7. Mount the Tape

Click LTFS at the top of the window
(you will see the name of the tape loaded) and select MOUNT*

*If this is a New LTO tape, the software will prompt you to format the tape.
*This step can usually be skipped since Yoyotta mounts the tape automatically when you Start the Job.


8. Add a Destination

Click the + sign on the right panel and select the mounted LTO tape. Double click it to add it as a destination.

9. Start the Job!

On the left panel (SOURCE), you should now see the Project name and the drive source. On the right panel (DESTINATION), you should see the Tape Code.

Click the Green Arrow at the top of the window to start the copy. 

10. Eject a Tape

Once you are done, eject the tape, clicking on the LTFS and selecting EJECT.
